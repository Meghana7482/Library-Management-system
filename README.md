# Library Management System

**Tech Stack:** Java, Object-Oriented Programming (OOP)

## Project Description
An OOP-based system to manage books in a library. Users can add, list, issue, and return books, with availability tracked for each book.

## Features
- Add new books to the library
- List all books with their availability status
- Issue and return books
- Simple command-line menu interface
